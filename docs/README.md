# Conexão Street v2

Site estático (GitHub Pages) + Supabase (Auth/DB/Storage/Realtime).

## Páginas
- index.html (home)
- products.html (planos)
- checkout.html (pix + pedido)
- member.html (área do membro)
- providers.html (listas após aprovação)
- quality.html / quality-gallery.html (Qualidade Fornecedores)
- admin.html (pagamentos/aprovações/qualidade)

## Deploy
1) Suba tudo no repositório
2) Ative GitHub Pages na branch principal
3) Rode os SQLs em /sql no Supabase
